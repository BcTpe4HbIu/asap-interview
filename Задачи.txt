Задачи:
Подготовить три среды: ru-dev, ru-prod, ru-staging

1. Для каждой среды создан кастомный конфиг nginx, нужно использовать его для каждой среды
2. Для каждой среды имя приложения должно использовать имя формата НАЗВАНИЕ_СРЕДЫ-НАЗВАНИЕ_АПИ
3. Для каждой объекта в среде должен быть добавлен лейбл env: НАЗВАНИЕ_СРЕДЫ
4. Для стейджа и прода использоваться одинаковые ресурсы
resources:
  limits:
    cpu: 300m
    memory: 300Mi
  requests:
    cpu: 100m
    memory: 100Mi
5. Для стейджа и прода использовать следующее количество реплик
replicas: 2
